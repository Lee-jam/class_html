# 211005 Responsive


### Viewport
------------
```
+ 반응형 웹 디자인은 HTML 및 CSS를 사용하여 웹 사이트의 크기를 자동으로 조정, 숨기기, 축소 또는 확대하여 모든 장치(데스크톱, 태블릿 및 휴대폰)에서 보기 좋게 만드는 것.
+ 반응형 웹사이트를 만들려면 <meta> 모든 웹페이지에 다음 태그를 추가해야함.
+ 이미지는 CSS width속성이 100%로 설정되면 이미지가 반응하고 확장 및 축소됨
+ 위의 이미지는 원래 크기보다 크게 확대될 수 있음. 많은 경우에 더 나은 솔루션은 width 대신 max-width속성을 사용하는 것.
+ max-width속성이 100%로 설정된 경우 이미지는 필요한 경우 축소되지만 원래 크기보다 크게 확대되지는 않음.
+ HTML <picture>요소를 사용하면 브라우저 창 크기에 따라 다른 이미지를 정의할 수 있음.
+ 브라우저 창의 크기를 조정하여 너비에 따라 아래 이미지가 어떻게 변하는지 확인함.
```
------------

### 반응형 텍스트 크기, 미디어 쿼리
------------
```
+ 텍스트 크기는 "뷰포트 너비"를 의미하는 "vw" 단위로 설정할 수 있음.
+ 뷰포트는 브라우저 창 크기. 1vw = 뷰포트 너비의 1%. 뷰포트의 너비가 50cm인 경우 1vw는 0.5cm
+ ex) <h1 style="font-size:10vw">Hello World</h1>
+ 텍스트와 이미지의 크기를 조정하는 것 외에도 반응형 웹 페이지에서 미디어 쿼리를 사용하는 것도 일반적임.
+ 미디어 쿼리를 사용하면 다양한 브라우저 크기에 대해 완전히 다른 스타일을 정의할 수 있음.
+ ex) @media screen and (max-width: 800px) {~~~~~}
```
